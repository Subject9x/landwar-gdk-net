LANDWAR Quick Play

Core Rules

Turn Order
  1. Initiative
  2. Movement
  3. Combat
  4. Damage
  5. Resolution
  • Determine Advantage for turn.
  • Move Units in Initiative Order.
  • Alternate making attacks.
  • Apply any damage from successful attacks.
  • Remove destroyed units, check end game.
note: where applicable, TAG rules override the core rules of the game.
1. Initiative
  • Panicked
    • any Units with Panicked token subtracts 1 from their Stress amount.
  • Players each roll 1D6.
  • Highest value Player becomes gainsAdvantage for turn.
    • assign +2 INI to a single non-panicked Unit.
  • Resolve any [TAG] rules for this phase.
2. Movement
  • all Units move in order of lowest MV to highest.
  • Advantage Player moves second on all ties of Move Value.
  • Move Types
    • Stationary: place token, Unit can rotate in place.
    • Normal: Unit may move up to its current total MV value.
    • TAG: make move as described in TAG rule.
  • Check Danger Close, Overwatch, mark any High Speed moves.
3. Combat
  • Any unit with Melee or Ranged values may make 1 attack.
  • Target must be closest within Line of Sight or suffer Stress for missed attacks.
  • Check Line of Sight: Enemy units block LoS, terrain may block Line of Sight.
  • Ranged Attack: check for Effective Range vs Long Range
  • Melee Attack: 0-1" range and facing enemy.
  • Basic ATK dice pool is 3.
  • Basic DEF dice pool is 3.
Combat Modifiers

ATTACK

  • Stationary +1 ATK
  • Panicked -3 ATK, -3 DEF to minimum of 1.
  • High Speed -2 ATK
  • Overwatch -2 ATK

DEFENSE

  • Long Range +1 DEF
  • Moved more than 1/4 MV - may spend Evade points on DEF rolls.
  • High Speed: re-roll 2 DEF.
  • Cover - may re-reroll half base DEF dice.
4. Damage
  • Each attack the Attacker and Defender roll their D6 pools.
  • Attacker counts each roll of 4+
  • Defender counts each roll of 5+
  • When Attacker success total is greater than Defender total, Target Unit takes all the Damage from the attack.
  • subtract Damage from Target's Armor stat.
  • Ranged Attack: missed shots at non-closest Target +1 stress at Effective Range OR +2 at Long Range.
5. Resolution
  • Any Units reduced to 0 Armor in the Damage phase are now marked as Destroyed.
  • Anytime a Player loses 1 or more Units in a Turn, all Units within 6" suffer +1 Stress.
  • Any Units with more than 0 Stress tokens must make a Stress Check.
  • Any Units with a Panicked token must make a Stress Check.
Stress
  • +1 Stress if Unit took any damage when Armor was 50% or less.
  • +1 Stress if Unit took any damage when Armor was 75% or less.
  • +1 Stress if Unit was target of 3 or more attacks this Turn.
  • Any TAG rules that apply as well.
Stress Check
  • Each Unit with 1 or more Stress Tokens rolls 1D6:
  • If result greater than or equal to total Stress Tokens, remove all tokens.
  • If result less than total Stress Tokens, give unit 1 Panicked token.
